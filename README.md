# H&N Car Rental Service

- This is a school project from Hive Helsinki designed to train the use of APIs by creating a page that pulls information from an API with a selection of cars and displaying information about them.

- I decided to transform this task into a functional car rental page, where users can "rent" vehicles from our selection. This allowed me to use my creativity and graphic design and UI/UX design skills to bring to life a fully functional and pleasant to use page that would suit any high end car rental company.

- This project makes use of Golang, Javascript, CSS and HTML.

*To run it, clone the repository and on your terminal type* `go run .`
